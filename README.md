#  Advanced Tax Optimization System (India)

##  Overview

This project is a real-world tax optimization system built using Python.
It calculates tax based on Indian tax rules and helps users optimize their tax savings.

---

##  Features

* Old vs New Tax Regime Comparison
* Salary Breakdown (Basic, HRA, Bonus)
* HRA Exemption Calculation
* Property Handling (Rental Income + Loan Interest)
* Multiple Tax Deductions (80C, 80D, NPS, 80E, 80G)
* Expense Analysis
* Smart Tax Saving Suggestions

---

##  Algorithms Used

* Merge Sort → Prioritizing deductions
* Greedy Algorithm → Maximizing tax savings
* Structured Tax Slab Computation

---

##  Workflow

1. Input salary details
2. Apply HRA exemption
3. Add property details
4. Apply deductions
5. Compare tax regimes
6. Analyze expenses
7. Suggest tax-saving strategies
